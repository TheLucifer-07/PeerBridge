# 🚀 PeerBridge

**PeerBridge** is a collaborative learning platform that helps students connect with peers, share academic resources, participate in subject discussions, and book tutoring sessions — all in one place.

Built during a hackathon to solve the problem of unstructured peer learning among students.

# 🎯 Problem

Students often struggle to:

- Find **reliable peer support** for difficult subjects  
- Organize **study materials shared across multiple platforms**  
- Schedule **structured tutoring sessions**  
- Participate in **focused academic discussions**

PeerBridge provides a **centralized and structured platform** to solve these challenges.

# 🌟 Features

## 🏠 Landing Dashboard
- Modern **hero section**
- Platform **statistics**
- **Trending subjects**
- **Student reviews**

## 👨‍🎓 User Dashboard
- Personalized **welcome section**
- **Live student statistics**
- **Available tutors preview**
- **Active discussion topics**

## 📚 Resources Section
- **Subject-wise filtering**
- **Search functionality**
- **Ratings and download counts**
- **Clean card-based UI**

## 👨‍🏫 Tutors Section
- List of **peer tutors**
- **Subject expertise**
- **Ratings**
- **Book session option**

## 💬 Discussions Section
- **10+ subject discussion rooms**
- **Chat-style interface**
- Messages stored in **LocalStorage**
- **Interactive user experience**

# 🛠 Tech Stack

- React.js
- Tailwind CSS
- JavaScript (ES6)
- Heroicons
- LocalStorage
- Custom SPA navigation using **History API**
